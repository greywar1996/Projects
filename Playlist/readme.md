# Работа с плейлистами — Тестирование REST API

**Описание:**

Учебный проект по тестированию REST API сервиса управления пользовательскими плейлистами.

**Цель проекта — отработать сценарное тестирование сущности "Playlist" с полным жизненным циклом:**

- создание
- изменение
- добавление элементов
- удаление
- проверка удаления

**Используемые инструменты:**

- Postman
- REST API
- Basic Authentication
- JavaScript (Postman test scripts)

**Авторизация:**

Используется Basic Auth на уровне коллекции.  
Логин и пароль вынесены в переменные окружения.

**Реализованные сценарии:**

### 1. Работа с плейлистом

- Создание кастомного плейлиста
- Получение информации о плейлисте
- Редактирование плейлиста
- Удаление плейлиста
- Проверка недоступности после удаления (404)

### 2. Работа с видео внутри плейлиста

- Добавление одного видео
- Добавление нескольких видео
- Получение списка видео
- Удаление видео из плейлиста

### 3. Негативные сценарии

- Повторное добавление уже добавленных видео
- Проверка корректных кодов ошибок (400 / 409)

**Проверяется:**

- HTTP статус-коды
- Структура ответа
- Наличие обязательных полей
- Корректная работа переменных
- Жизненный цикл сущности
- Бизнес-ограничения (дублирование видео)

**Как запустить:**

1. Импортировать коллекцию в Postman
2. Создать Environment
3. Указать:
   - base_url
   - username
   - password
4. Выполнять запросы в логической последовательности

**Результат:**

Проект демонстрирует:

- понимание REST архитектуры
- умение строить сценарные тесты
- работу с вложенными ресурсами
- тестирование бизнес-ограничений
- управление зависимыми запросами через переменные
