# RVR-1 — Тестирование REST API

**Описание:**

Учебный проект по тестированию REST API.

**Цель — отработка:**

- позитивных и негативных сценариев
- проверки HTTP статус-кодов
- базовой валидации ответов API
- работы с переменными в Postman

**Используемые инструменты:**

- Postman
- REST API
- JavaScript (Postman test scripts)

**Реализовано:**

- Проверка корректных статус-кодов
- Позитивные сценарии
- Негативные сценарии
- Проверка структуры ответа
- Работа с переменными окружения

**Как запустить:**

1. Импортировать коллекцию в Postman
2. Создать Environment
3. Указать base_url
4. Выполнять запросы

**Результат:**

Проект демонстрирует:

- базовое понимание тестирования REST API
- умение строить позитивные и негативные проверки
- работу с тест-скриптами в Postman
