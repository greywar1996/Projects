# DummyJSON API Testing Project

## О проекте

Тестирование публичного REST API DummyJSON.

## Структура проекта

- 'collection/' - Postman Collection + API + SQL + readme
- 'testcases/' - тест-кейсы + readme
- 'bugreports/' - баг-репорты + readme
- 'sql/' - структура БД + readme

## Результат

Проект демонстрирует:

- понимание работы REST API
- умение находить дефекты
- системный подход к тестированию
- аккуратность в оформлении тестовой документации
